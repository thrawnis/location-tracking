Import from Google Takeout

Bring your saved places over from Google Maps:

  1. Go to takeout.google.com
  2. Deselect all, then select only Saved (or Maps (your places))
  3. Export, download the archive, and find Saved Places.json
  4. Upload it below — generic GeoJSON files work too

Places whose name already exists are skipped, so re-importing is safe.
